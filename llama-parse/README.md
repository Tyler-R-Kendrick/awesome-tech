# LlamaParse Cloud Index v2 Setup

## Required Setup

Use this official guide for setup and usage:

- https://developers.llamaindex.ai/llamaparse/cloud-index-v2/getting_started/

## Repository Policy

- Use Cloud Index v2 for repository document parsing and indexing.
- Do not add alternative parsing/indexing paths that bypass this setup.
- Ensure indexed content conforms to:
  - LLM Wiki 2.0 (LangChain standard)
  - Open Knowledge format (Google standard)
