# x402 Payment Protocol

An open payment protocol that revives the HTTP 402 "Payment Required" status code so clients — including AI agents — can pay for API and content access programmatically over HTTP, without accounts or sessions, using stablecoins and other value rails. Created and open-sourced by Coinbase (May 2025) and contributed to the Linux Foundation's x402 Foundation in April 2026.
