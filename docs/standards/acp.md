# Agent Communication Protocol (ACP)

An open protocol from IBM Research (launched March 2025) for communication between AI agents, applications, and humans, originally the interoperability layer for the open-source BeeAI platform. In August 2025 ACP converged with the Agent2Agent (A2A) project under the Linux Foundation; independent development wound down and its technology folded into A2A.

## Footnotes

- Do not confuse this with Anthropic's unrelated "Agent Client Protocol" (also abbreviated ACP), which connects code editors to agents.
- ACP is effectively in maintenance as an independent standard; new development is directed to A2A.
