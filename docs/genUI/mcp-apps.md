# MCP Apps

The official Model Context Protocol extension (proposal SEP-1865) that lets MCP servers return interactive HTML UI resources rendered inline in MCP clients such as Claude and ChatGPT.

## Contents

- [modelcontextprotocol/ext-apps](https://github.com/modelcontextprotocol/ext-apps) - Official spec and SDK for the MCP Apps protocol.
- [MCP Apps extension overview](https://modelcontextprotocol.io/extensions/apps/overview) - Extension documentation.
- [MCP-UI-Org/mcp-ui](https://github.com/MCP-UI-Org/mcp-ui) - Community SDK for building interactive UI resources over MCP (TypeScript, Ruby, Python).

## Footnotes

- Tool-owned UI: the server returns UI resources that render inline in compatible hosts.
