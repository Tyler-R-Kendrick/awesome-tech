# OpenUI

"OpenUI" is used by two distinct generative-UI projects; both are tracked here so the reference is unambiguous.

## Contents

- [wandb/openui](https://github.com/wandb/openui) - Weights & Biases tool to describe a UI in natural language, preview it live, and convert the result to React, Svelte, or Web Components.
- [thesysdev/openui](https://github.com/thesysdev/openui) - Thesys's generative-UI framework built around "OpenUI Lang," a compact streaming-first language plus a React runtime.

## Footnotes

- The widely-cited "OpenUI" is `wandb/openui`; `thesysdev/openui` is a separate, similarly named framework (Thesys also ships the hosted C1 Generative UI API).
