# Context Engineering

A curated index of guides, papers, and tools for deliberately designing, selecting, and managing the information placed in an LLM or agent context window.

## Contents

- [Effective context engineering for AI agents](https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents) - Anthropic's guide to curating system prompts, tools, examples, and history within an agent's finite context.
- [The New Skill in AI is Context Engineering](https://www.philschmid.de/context-engineering) - Phil Schmid's framing of context engineering as building dynamic systems that supply the right information and tools at the right time.
- [Context Engineering for Agents](https://blog.langchain.com/context-engineering-for-agents/) - LangChain's categorization of agent context strategies into writing, selecting, compressing, and isolating context.
- [Lessons from Building Manus](https://manus.im/blog/Context-Engineering-for-AI-Agents-Lessons-from-Building-Manus) - Practical context-engineering lessons, including KV-cache optimization and tool masking, from building the Manus agent.
- [A Survey of Context Engineering for Large Language Models](https://arxiv.org/abs/2507.13334) - Academic survey systematizing context engineering across retrieval, memory, and multi-agent systems.
- [davidkimai/Context-Engineering](https://github.com/davidkimai/Context-Engineering) - First-principles handbook and notebook collection for moving beyond prompt engineering to context design.
- [12-Factor Agents](https://github.com/humanlayer/12-factor-agents) - Engineering principles for reliable LLM applications, including owning your prompts and your context window.
- [langchain-ai/context_engineering](https://github.com/langchain-ai/context_engineering) - Companion LangGraph notebooks implementing the write, select, compress, and isolate context strategies.
