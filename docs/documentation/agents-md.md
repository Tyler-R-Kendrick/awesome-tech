# AGENTS.md

A simple, open format for guiding coding agents: a dedicated, predictable file where a project provides the build steps, tests, conventions, and context that AI coding agents need to work effectively. It complements human-focused README files — keeping README content concise while giving agents a single place to look for machine-relevant instructions. AGENTS.md emerged from collaboration across the AI software development ecosystem (including OpenAI Codex, Amp, Jules by Google, Cursor, and Factory) and is stewarded by the Agentic AI Foundation under the Linux Foundation.

## Contents

- [agents.md](https://agents.md) - Official site, format overview, and examples.
- [github.com/agentsmd/agents.md](https://github.com/agentsmd/agents.md) - Project repository.

## Footnotes

- AGENTS.md is an agent-facing file format in the same family as [Agent Skills](../agents/skills/README.md) (`SKILL.md`), [design.md](./design-md.md), and the [Open Knowledge Format](./okf.md).
- This repository follows the convention: each folder scope carries an `AGENTS.md` with folder-specific guidance.
