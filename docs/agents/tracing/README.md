# Agent Tracing

A curated index of observability standards and tools for tracing agent execution.

## Contents

- OpenInference - Observability standard for capturing AI and agent execution traces.
- OTEL Gen AI standards - OpenTelemetry conventions and standards for generative AI telemetry.
- [LangSmith](./langsmith.md) - LangChain's framework-agnostic platform for tracing, evaluating, and deploying agents, with native and OpenTelemetry instrumentation.
- [LangSmith Engine](./langsmith-engine.md) - Deep agent that clusters production trace failures into issues and drafts PRs, evaluators, and eval datasets for review.
